# AcmeCorp Network Project

A multi-phase enterprise network built in Cisco Packet Tracer as part of my CCNA 200-301 study and portfolio.

## Scenario
AcmeCorp is a company with:
- HQ in Sydney
- Branch 1 in Melbourne
- Branch 2 in Brisbane
- An ISP connecting all sites
- A cloud/data centre reachable over the internet
